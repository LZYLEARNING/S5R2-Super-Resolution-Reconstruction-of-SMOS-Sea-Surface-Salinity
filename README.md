# S5R2-Super-Resolution-Reconstruction-of-SMOS-Sea-Surface-Salinity
S5R2: SMOS satellite sea surface salinity super-resolution reconstruction network, which achieved super-resolution reconstruction of the SMOS L3 SSS product from 1/4° to 1/12° by fusing multivariate satellite observations. Doi: 10.1109/JSTARS.2025.3602684.
